About
В качестве СУБД для проекта используется H2.
В качестве веб-сервера (контейнера сервлетов), в проекте используется Jetty.
По умолчанию сервис поднимается на порту 9999.

________________

Развернуть проект
 Для развертывания проекта используя среду разработки IntelliJ IDEA требуется создать
конфигурацию Maven, в строке Command Line указать clean flyway:clean flyway:migrate package
jetty:run. Аналогичные команды можно ввести в командную строку IDE.
Команды flyway:clean и flyway:migrate имеет смысл использовать только для первого
запуска, т.к. они соответственно удаляют(очищают) и создают заново базу данных и заполняют ее
данными.

________________

Запросы для создания базы
В проекте используется flywaydb  для создания базы и справочника, и заполнения их данными.
Файлы миграций расположены ./src/main/resources/db/migration.
